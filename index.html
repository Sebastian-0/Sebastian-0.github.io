Survival Isle

This is a game created as a project in the course EDA095 - Nätverksprogrammering on LTH. The game is a multiplayer 2D survival game where all players must cooperate to survive as long as possible.

How to play

First you must download the game client using the link on the top of the site. To start the game you can double click on the jar file or run the game from the console using the following command:

    java -jar Client.jar
From the menu can connect to a server by entering a server ip-address and port number.

If you wish to host a server on your own you can also download the server application from the top of this site. To start a server you can use the console and enter the following command

    java -jar Server.jar port_number
Here port_number is the chosen port number each client need to enter to be able to connect.

About our project

The game was written in Java together with the OpenGL library libgdx for graphical support. The game uses TCP-sockets to send and recieve data between the server and the client. The client and server also uses a protocol we defined to easier recieve and send information.
